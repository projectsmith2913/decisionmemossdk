# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email: **security@decisionmemos.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- SDK version and Node version

You will receive an acknowledgement within 48 hours. We aim to release a fix within 14 days of a confirmed report.

We do not currently offer a bug bounty programme, but we will credit researchers in the changelog unless you prefer to remain anonymous.
